How to run the application:

1. In the folder where you kept the java files, open your terminal and run the command "javac *.java"
2. After compiling, in your terminal, run the command "java Main"

Please take note of the following
	- Ensure that all the Java and text files are in the same folder
	- Ensure that the resources folder is in the same folder as the Java files.
	- Do NOT alter the folder structure of the application files.
	- Do NOT change any of the filenames in the application.
	- The terminal commands are case-sensitive
